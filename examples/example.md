# Пример документа

Это тестовый Markdown документ для проверки конвертера Fura.

## Основные возможности

Поддержка стандартной Markdown разметки:

- Списки
- **Жирный текст**
- *Курсив*
- `Код`

### Пример кода

```javascript
function hello() {
  console.log('Hello, Fura!');
}
```

## Заключение

Простой конвертер Markdown в PDF формат A4.
